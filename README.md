# PlantLens - Landing Page Conceitual 🌿

Esta é a **Landing Page** do *PlantLens*, um projeto acadêmico de apresentação de um aplicativo teórico (fictício) para identificação e cuidado de plantas. O site em si foi desenvolvido puramente como um exercício de front-end, focado em criar a interface de um produto digital em um ambiente web. O aplicativo descrito na página, assim como as funcionalidades de Inteligência Artificial, não existem na realidade.

🔗 **Acesse o Site (Design Responsivo):** [PlantLens na Vercel](https://plantlens-landing-page.vercel.app)

## 📝 Sobre o Projeto

Desenvolvido em equipe para o Curso de Desenvolvimento de Sistemas, o desafio foi estruturar uma página web atraente capaz de "vender" o conceito de uma ferramenta que resolve problemas reais. É um estudo de caso focado no layout, fluxo de informações e experiência do usuário (UX/UI) na web.

## 💻 Destaques Técnicos da Landing Page

O projeto focou no desenvolvimento da interface web abordando os seguintes aspectos:
- **Design de Produto:** Apresentação clara de missão, recursos teóricos (IA, banco de dados botânico, rotinas) e diferenciais do suposto app.
- **Responsividade:** Layout adaptável focado na boa visualização tanto em dispositivos móveis quanto em desktops.
- **Interface Intuitiva:** Criação de uma navegação fluida para simular a apresentação comercial de um aplicativo de ponta.

## 👥 Equipe de Desenvolvimento

Este projeto prático de elaboração de Website (e criação do conceito e identidade visual da aplicação) foi idealizado e construído de forma colaborativa, consolidando o aprendizado no curso de Desenvolvimento de Sistemas.
